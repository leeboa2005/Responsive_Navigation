# Responsive_Navigation

### 소개

반응형 네비게이션기능이 있는 웹페이지입니다.

<br>

### 기능

- 반응형으로 제작한 웹페이지 입니다.
- PC, 테블릿, 모바일에 맞게 네비게이션이 유동적으로 변경됩니다.

<br>

### 폴더구조

```
📦 Responsive_Navigation
┣ 📂js
┃ ┗📜main.js
┣ 📂style
┃ ┣📜commmon.css
┃ ┗📜reset.css
┗ 📜index.html
```

<br>

### 파일설명

- main.js : 모바일 화면에서 보이는 햄버거 버튼을 클릭 했을때 토글 기능을 수행 하게 해줍니다.
- reset.css : css 초기화 코드
- common.css : 전체 css
- index.html : 전체적인 구조가 담겨있습니다.
